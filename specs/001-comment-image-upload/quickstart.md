# Quickstart: コメント画像アップロード

## Prerequisites

- Node.js / pnpm available
- VS Code拡張の開発環境が有効

## Setup

1. `pnpm install`
2. `pnpm run lint`
3. `pnpm test`

## Feature Validation (manual)

1. コメントにローカル画像のMarkdownリンクを含めて保存する
2. 保存後に画像が表示されることを確認する
3. 画像リンクが不正な場合は保存が失敗することを確認する
4. 画像リンクを含まないコメント保存が従来通りであることを確認する
