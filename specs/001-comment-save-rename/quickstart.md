# Quickstart: Comment Save Rename

## Goal

新規コメント保存後にエディタが更新モードへ移行し、同一コメントを継続編集できることを確認する。

## Scenarios

1. **新規コメント保存→更新モード移行**
   - コメント追加エディタを開く
   - コメントを保存
   - エディタが更新モードに切り替わることを確認（ファイル名は変更しない）
   - そのまま再保存して同一コメントが更新されることを確認

2. **保存失敗時の挙動**
   - コメント保存を失敗させる（通信失敗など）
   - 追加モードのままであることを確認
   - 再保存成功時に更新モードへ切り替わることを確認

3. **識別子取得失敗時の挙動**
   - コメント保存後に識別子取得が失敗するケースを再現
   - 追加モードのまま維持され、通知が表示されることを確認

4. **保存直後にエディタを閉じる**
   - 保存直後にエディタを閉じる
   - 更新モードへの切り替えがバックグラウンドで完了し通知されることを確認

5. **複数エディタ同時利用**
   - 複数の新規コメントエディタを開く
   - 1つを保存
   - 保存対象エディタのみが更新用に切り替わることを確認
