# Quickstart: Project List Sidebar

## Prerequisites

- A Redmine 6.1 instance URL.
- A personal API key with access to target projects.

## Configure

1. Open the extension settings and provide your Redmine base URL.
2. Add your API key for authentication.

## Browse Projects

1. Open the "Redmine Projects" view in the sidebar.
2. Select a project to highlight it and load tickets.
3. The last selected project is remembered on startup.
4. Use "Redmine: Refresh Projects" if the list fails to load.

## Browse Tickets

1. Open the "Redmine Tickets" view in the sidebar.
2. Confirm tickets reflect the selected project.
3. If the project has no tickets, an empty-state message is shown.

## Error Handling

- If project loading fails, the extension shows an error and offers retry.
