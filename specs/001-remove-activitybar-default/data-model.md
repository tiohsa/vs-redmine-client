# Data Model

## Entity: メタデータ既定値定義
- **目的**: ファイルで管理される既定値の集合
- **主な属性**:
  - **定義ファイルパス**: 既定値の参照元
  - **既定値マップ**: メタデータ項目ごとの既定値
- **検証ルール**:
  - 既定値が未定義の場合は空として扱う

## Entity: チケットメタデータ
- **目的**: チケットに付随する項目群と初期値
- **主な属性**:
  - **メタデータ項目**: キーと値の集合
  - **初期化状態**: 既定値適用済み/空
- **検証ルール**:
  - 既定値定義がある場合は一致する初期値を設定する
  - 既定値定義がない場合は空の状態で開始する

## Relationships
- チケットメタデータはメタデータ既定値定義を参照して初期化される
