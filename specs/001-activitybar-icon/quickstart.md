# Quickstart: Activity Bar Icon Update

## Preconditions
- Install and launch the TodoEx extension in VS Code.

## Manual Verification
1. Open VS Code and locate the TodoEx Activity Bar icon.
2. Confirm the icon shape matches the provided SVG path.
3. Switch between light and dark themes and verify the icon color follows the theme.

## Automated Tests
- Add or update a unit test to verify `media/todoex-activitybar.svg` contains the new SVG path.
