# Data Model: エディタ初期値設定

## Entity: 初期値設定

- **Purpose**: 新規登録エディタの入力項目に対する既定値を保持する
- **Identity**: 利用者ID（同一利用者内で一意）
- **Fields**:
  - **userId**: 利用者を識別するID
  - **defaults**: 項目キーと初期値のマップ
  - **updatedAt**: 最終更新時刻

## Entity: 新規登録エディタ項目

- **Purpose**: 新規登録時に編集可能な入力項目
- **Identity**: 項目キー（エディタ内で一意）
- **Fields**:
  - **fieldKey**: 項目識別子
  - **label**: 表示名
  - **validationRules**: 入力規則（必須/形式/長さなど）

## Relationships

- **初期値設定** 1 --- 0..* **新規登録エディタ項目**
  - 初期値設定は項目キーで各項目に紐づく

## Validation Rules

- 初期値は各項目の入力規則に従うこと
- 無効な初期値は保存できず、エラー表示されること
- リセットは該当項目の初期値を空欄として保存すること

## State Transitions

- 未設定 → 設定済み
- 設定済み → 変更済み
- 設定済み → リセット済み（空欄保存）
