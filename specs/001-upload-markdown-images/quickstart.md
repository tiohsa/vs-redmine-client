# Quickstart: Markdown画像リンクの自動アップロード

## Prerequisites

- Redmine接続設定が完了していること。
- 「添付追加」権限を持つユーザーであること。

## チケット作成/編集での自動アップロード

1. チケット作成または編集の本文にMarkdown画像リンクを記載する。
2. 保存するとローカル画像がアップロードされ、リンクが置き換わる。
   - 外部URLの画像リンクは対象外。

## コメント追加/編集での自動アップロード

1. コメント追加または編集の本文にMarkdown画像リンクを記載する。
2. 保存するとローカル画像がアップロードされ、リンクが置き換わる。
   - 相対パスは編集中のファイル位置を基準に解決される。

## Error Handling

- 添付権限がない場合は保存は継続され、アップロード不可が通知される。
- 画像のアップロードに失敗しても本文は保存され、失敗一覧が表示される。
- 対応形式はPNG/JPEG/GIF/WebP、サイズは1枚10MB以下。
